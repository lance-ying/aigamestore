Block Blast!

Block Blast! is a puzzle game where you place geometric blocks on a 9x9 grid. Your goal is to fill entire rows or columns to clear them and earn points. You have three blocks available at a time, and must strategically place them to maximize your score. The game ends when you can't place any of the available blocks on the grid. Earn combo bonuses by clearing multiple lines simultaneously or consecutively!

Arrow Left/Right: Move the selected block horizontally
Arrow Up/Down: Move the selected block vertically
Z: Cycle through available blocks
Space: Place the selected block on the grid
Enter: Start the game from the start screen
Esc: Pause/unpause the game
R: Restart the game when game is over
