Royal Kingdom

Welcome to Royal Kingdom! Match colorful tiles to complete challenging objectives across 5 progressive levels. Create matches of 3 or more identical tiles by swapping adjacent pieces. Form matches of 4 tiles to create powerful Rockets that clear entire rows or columns, or match 5+ tiles to generate Bombs that explode in a 3x3 area! Break through Ice Blocks, destroy Chained Tiles, and collect special Target Items to complete each level's objectives before running out of moves. Plan your moves carefully and create cascading matches for bonus points!

Arrow Keys: Move cursor to select tiles
Space: Select tile / Swap with selected tile / Activate special piece
WASD: Alternative movement keys (W=Up, A=Left, S=Down, D=Right)
Enter: Start game from menu
Esc: Pause / Unpause game
R: Return to start screen (from game over)
