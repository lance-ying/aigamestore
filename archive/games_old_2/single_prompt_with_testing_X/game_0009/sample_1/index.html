SPILL IT!

Drop balls to topple glasses in this physics-based puzzle game! Each level presents a set of glasses that you must knock over by strategically dropping a limited number of balls. You need to knock over at least 80% of the glasses to progress to the next level. Complete levels to unlock new ball skins and themes. The challenge increases with more glasses, obstacles, and trickier arrangements!

Controls: LEFT/RIGHT ARROW or A/D: Move ball drop position | SPACE: Drop a ball | ENTER: Start game | ESC: Pause/Unpause | R: Restart to title screen
