Track Runner 3D

Run endlessly through a futuristic train station! Switch between three parallel tracks to dodge oncoming trains, jump over barriers, and slide under obstacles. Collect golden coins to increase your score while the game speeds up progressively. Survive as long as possible to achieve the highest score!

A / Left Arrow: Move to the left track | D / Right Arrow: Move to the right track | W / Up Arrow: Jump over obstacles | S / Down Arrow: Slide under barriers | ENTER: Start the game | ESC: Pause the game | R: Restart after game over
