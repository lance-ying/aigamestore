Farkle Fortune

Farkle Fortune is a strategic dice game where you compete against an AI opponent to be the first to reach the target score. Roll six dice, select scoring combinations, and decide whether to bank your points or risk rolling again for more. Watch out for Farkles - rolling no scoring dice loses all points for that turn! Progress through three increasingly challenging levels with rising target scores and smarter AI opponents.

Arrow Keys: Navigate dice
Space: Roll dice / Roll again
Shift: Select/deselect die
B: Bank score (when minimum reached)
Z: Deselect all dice
Enter: Start game / Continue
ESC: Pause / Unpause game
R: Restart game (from game over)
