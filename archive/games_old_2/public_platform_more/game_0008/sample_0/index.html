Hitman: Sniper

You are Agent 47, the world's deadliest sniper. Eliminate high-value targets from your fixed vantage point overlooking a secure compound. Use precision aim and environmental kills to complete objectives before time runs out. Stay undetected—if guards spot you, the mission fails. Score multipliers reward stealth and accuracy. Complete missions to unlock better weapons and tackle the zombie survival mode.

Arrow Keys: Move crosshair/scope aim | Space: Fire weapon | Shift: Toggle zoom (2x/4x magnification) | Z: Reload weapon | ESC: Pause/Unpause game | R: Return to start screen (from game over) | ENTER: Start game (from start screen)
