DEUS EX GO

Guide Adam Jensen through a dangerous grid-based facility to reach the exit. This is a turn-based stealth puzzle game where every move you make triggers enemy movement. Avoid enemy line-of-sight, hack terminals to disable turrets, and use augmentations like temporary invisibility to survive. Plan your moves carefully - enemies patrol in patterns, and getting caught means game over. Complete increasingly challenging levels with new enemy types and obstacles.

Arrow Keys: Move Adam Jensen one grid space | Space: Activate temporary invisibility augmentation (1 turn, limited uses) | Z: Hack adjacent terminal to toggle turrets on/off | Shift: Wait (skip turn without moving) | Enter: Start game | Esc: Pause/Unpause | R: Restart and return to START screen
