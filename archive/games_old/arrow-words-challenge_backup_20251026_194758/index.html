Arrow Words Challenge

Arrow Words Challenge is a crossword-style puzzle game where you fill in a grid of words using letter clues embedded directly in the grid. Each clue has an arrow pointing right or down to indicate the direction of the word. Complete all words correctly to win the level. Use hints wisely as they cost points. Progress through 5 increasingly challenging levels, each with larger grids and more complex words. Earn points for correct letters, completed words, and fast completion times!

ENTER: Start game from menu | ESC: Pause/unpause game | R: Return to start screen | Arrow Keys: Navigate between cells in the grid | A-Z: Type letters into selected cells | BACKSPACE: Delete letter and move back | SPACE: Jump to next empty cell in current word | TAB: Use hint to reveal a letter (costs points)
