Bring You Home

Guide Polo safely home by rearranging the world around him! Each level consists of several panels containing paths, obstacles, and hazards. Your job is to drag and swap these panels to create a safe route from start to exit. Polo walks automatically along the path you create, so plan carefully! Avoid spikes, enemies, and gaps. If Polo meets danger, use the rewind button to try again. Complete all levels in each world to progress to harder challenges with more complex panel arrangements and new environmental mechanics.

Arrow Keys (←/→): Select different panels to swap | Space: Swap the two selected panels | Z: Start Polo walking on your arranged path | Shift: Rewind Polo to the start if he's in danger | Enter: Start the game from the title screen | Esc: Pause/Unpause the game | R: Restart and return to the title screen
