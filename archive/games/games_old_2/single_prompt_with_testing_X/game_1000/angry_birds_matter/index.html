Red's First Flight

Launch Red the bird using a slingshot to destroy structures and hit all the green pigs to complete each level. Pull back the slingshot to aim and release to launch. Each level has limited birds to use - plan your shots carefully! Destroy structures strategically to make them collapse onto pigs for maximum effect.

Controls: SPACE: Pull back slingshot / Release to launch | Arrow Keys (LEFT/RIGHT): Adjust aim angle | Arrow Keys (UP/DOWN): Adjust launch power | ENTER: Start game | ESC: Pause | R: Restart
