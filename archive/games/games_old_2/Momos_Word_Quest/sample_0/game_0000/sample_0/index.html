Momo's Word Quest

Help Momo learn English vocabulary! Progress through 5 challenging levels, each focusing on different word types and mini-games. Match images to words, choose correct definitions, spell words by typing, and complete sentences. Earn points for correct answers, build streaks for bonus points, and race against the clock. Each level gets progressively harder with new vocabulary and game mechanics. Can you master all levels and achieve the highest score?

ENTER: Start game from menu | ARROW KEYS: Navigate menu options and highlight answer choices | SPACE/ENTER: Confirm selection or submit answer | KEYS 1-4: Directly select answer option in multiple choice | KEYS A-Z: Type letters in spelling challenges | BACKSPACE: Delete letter when typing | SHIFT: Capitalize letters when typing | ESC: Pause/unpause game | R: Return to start screen from game over | H: Use hint (costs 100 points)
