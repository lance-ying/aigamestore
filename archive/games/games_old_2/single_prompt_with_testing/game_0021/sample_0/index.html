Dungeon Tracer

Navigate a procedurally generated dungeon by connecting adjacent tiles of the same type to attack enemies, defend against damage, and heal yourself. Connect 3+ matching tiles to activate their effect: swords attack nearby enemies, shields reduce incoming damage, and potions restore health. Defeat enemies to gain experience, level up, and increase your stats. Survive as long as possible and achieve the highest score!

Arrow Keys: Navigate menu selections | Space: Confirm tile selection and start/end path drawing | Shift: Cancel current path | Z: Quick heal (if available) | Enter: Start game from menu | ESC: Pause/Unpause game | R: Restart game (return to start screen)
