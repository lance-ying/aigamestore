Fruit Merge: Match Game

Drop and merge identical fruits to create bigger ones! Start with cherries and work your way up to the ultimate watermelon. Carefully position each fruit before dropping it into the container. When two matching fruits touch, they combine into the next fruit in the sequence. Keep all fruits below the danger line at the top - if any fruit crosses it for too long, it's game over! Score points with every merge and aim for the highest score possible.

Arrow Left/A: Move fruit left | Arrow Right/D: Move fruit right | Arrow Down/S: Drop fruit faster | SPACE: Quick drop fruit | ENTER: Start game | ESC: Pause game | R: Restart (from game over screen)
