Shoot The Zombirds

Defend your precious pumpkin patch from waves of mutant Zombirds! Fire crossbow bolts by pressing the arrow keys to aim and SPACE to shoot. Each wave brings more dangerous enemies with varying health and speeds. Earn coins to upgrade your crossbow between waves. Survive as long as possible - if all 5 pumpkins are destroyed, it's game over!

Arrow Keys (UP/DOWN/LEFT/RIGHT): Aim crossbow in that direction | SPACE: Fire crossbow bolt | SHIFT: Activate multi-shot power (when available) | Z: Activate shield power (when available) | ENTER: Start game from start screen | ESC: Pause/Unpause game | R: Restart (returns to start screen)
