Fling!

Clear the board until only one furball remains! Slide furballs in cardinal directions to push others off the edge. Each furball slides until hitting another furball or a wall. Plan your moves carefully - each puzzle has a unique solution. Progress through increasingly challenging levels with more furballs and complex layouts.

Arrow Keys: Select and move furballs (Up/Down/Left/Right) | Space: Confirm furball selection and execute move | Z: Undo last move | R: Reset current puzzle | Enter: Start game / Next level | Esc: Pause/Unpause game
