Number Surge

Number Surge is an action-puzzle runner where you control a numbered block racing forward through a challenging obstacle course. Absorb smaller numbered blocks to grow your value, but avoid larger numbers and deadly hazards! Navigate wisely to build the highest number possible, then use your accumulated power to smash through numerical walls at the end of each level. Complete all 5 levels to become the Number Master!

Arrow Left/Right: Move horizontally along the track | Space: Select menu options / Continue after level complete | Shift: Pause/Unpause during gameplay | Enter: Start game from main screen | Esc: Pause game | R: Restart (return to start screen)
