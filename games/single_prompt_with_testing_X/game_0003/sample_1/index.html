Jelly Shift - Obstacle Course

Navigate your jelly character through an endless obstacle course! Change your jelly's shape between tall and short to squeeze through gaps and barriers. Collect diamonds for points and build combos by avoiding obstacles to enter Jelly Fever mode for a speed boost. Reach the finish line of each level to progress. The game gets progressively harder with faster speeds and more complex obstacles.

UP ARROW: Change jelly to tall/thin shape | DOWN ARROW: Change jelly to short/wide shape | ENTER: Start game from start screen | ESC: Pause/unpause game | R: Return to start screen
