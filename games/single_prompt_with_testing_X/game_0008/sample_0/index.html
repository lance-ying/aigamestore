Snake Vs Block

Control a snake made of numbered balls as it automatically moves forward through an endless vertical track. Collide with numbered blocks to reduce the snake by that number, or collect floating balls to grow longer. The goal is to survive as long as possible by maintaining your snake's length above zero while traveling the maximum distance. Blocks increase in value as you progress, making survival increasingly challenging.

Controls:
Arrow Left / A: Steer snake left
Arrow Right / D: Steer snake right
ENTER: Start game
ESC: Pause/Unpause
R: Restart (from game over screen)
