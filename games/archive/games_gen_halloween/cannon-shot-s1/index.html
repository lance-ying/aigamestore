Cannon Shot!

Guide cannon balls into target buckets by strategically placing ramps and bumpers! Each level gives you a limited number of redirecting objects to position before firing 20 balls from the cannon. Your goal is to get the required number of balls into each colored bucket to complete the level. Plan carefully - once you fire, physics takes over! Complete levels to unlock new cannon designs and face increasingly challenging puzzles.

Arrow Keys: Move selected object (when in placement mode)
Space: Fire cannon (when ready) / Reset level (after firing)
Z: Rotate selected object counterclockwise
X: Rotate selected object clockwise
A/D: Cycle through available objects to place
Enter: Start game from title screen
Esc: Pause/Unpause game
R: Return to start screen
