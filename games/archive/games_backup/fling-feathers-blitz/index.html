Fling Feathers Blitz

Fling Feathers Blitz is a physics-based puzzle game where you launch birds from a slingshot to destroy structures and eliminate all pigs on each level. Master the unique abilities of different bird types - Red birds are basic projectiles, Blue birds split into three, and Yellow birds boost their speed mid-flight. Complete all 5 levels by strategically using your limited birds to topple wooden and stone structures, earning points for destruction and bonus points for unused birds. Each level increases in difficulty with more complex structures and better-protected pigs!

ENTER: Start the game from the start screen
SPACE: Press once to aim, press again to launch the bird
Arrow Keys (UP/DOWN/LEFT/RIGHT): Adjust slingshot aim while aiming
Z: Activate the current bird's special ability while in flight
ESC: Pause or unpause the game
R: Restart and return to the start screen
