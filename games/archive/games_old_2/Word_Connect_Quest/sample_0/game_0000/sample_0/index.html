Word Connect Quest

Welcome to Word Connect Quest! Form words by connecting letters in the circular wheel to fill the crossword grid. Each level presents a new puzzle with hidden words waiting to be discovered. Complete all words in a level to advance. Beat the target time for bonus points and aim for the highest score across all three levels!

ENTER: Start the game from the main menu | ESC: Pause/unpause during gameplay | R: Restart and return to the main menu | Arrow Keys: Navigate through letters in the wheel (alternative to mouse) | Space: Select/deselect letter when using keyboard navigation | Z: Submit word when using keyboard navigation
