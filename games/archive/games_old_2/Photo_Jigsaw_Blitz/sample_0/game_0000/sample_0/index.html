Photo Jigsaw Blitz

Photo Jigsaw Blitz is a fast-paced puzzle game where you assemble fragmented photo pieces into complete images against the clock. Select and move puzzle pieces around the canvas to align them correctly. When two pieces are close enough and properly aligned, they'll automatically snap together. Chain multiple snaps for bonus points! Complete 4 increasingly challenging levels, each with more pieces and tighter time limits. Race against time to complete the masterpiece!

ENTER: Start game from the start screen | SPACE: Select/deselect the nearest puzzle piece or group | Arrow Keys: Move selected piece/group (normal speed) | SHIFT + Arrow Keys: Move selected piece/group (fine-tuned, slower speed) | Z: Rotate selected piece/group 90° counter-clockwise (enabled in levels 2-4) | ESC: Pause/unpause game | R: Restart and return to start screen
