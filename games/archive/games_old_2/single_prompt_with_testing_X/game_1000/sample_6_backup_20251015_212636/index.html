Bird Slingshot Physics

Launch birds from a slingshot to destroy enemy structures and defeat all pigs to win! Each bird has a unique special ability that can be activated mid-flight by pressing SPACE. Aim carefully and use physics to your advantage - ricochet off surfaces, cause chain reactions, and topple towers. Three lives per level - make each shot count!

Arrow Keys (LEFT/RIGHT): Aim the slingshot angle | Arrow Keys (UP/DOWN): Adjust launch power | SPACE: Launch bird (when aiming) / Activate bird's special ability (mid-flight) | ENTER: Start game from start screen | ESC: Pause/Unpause game | R: Restart to start screen from game over
