Bubble Cash

Pop and blast colored bubbles in this fast-paced bubble shooter! Match three or more bubbles of the same color to clear them from the board. You're competing against AI opponents in a tournament-style match where speed and accuracy matter. Clear all bubbles before time runs out to achieve the highest score and rank in the top 3 to win!

LEFT ARROW: Aim laser guide left | RIGHT ARROW: Aim laser guide right | SPACE: Launch bubble | ENTER: Start game from start screen | ESC: Pause/Unpause game | R: Return to start screen
