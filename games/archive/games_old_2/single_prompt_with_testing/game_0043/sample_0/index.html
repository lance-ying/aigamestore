Aventureros al Tren®

Build railway routes across North America in this turn-based strategy game! Collect colored train cards and claim routes between cities to score points. Complete destination tickets to earn bonus points by connecting specific cities. The game ends when a player has 3 or fewer trains left, triggering the final round. Win by having the highest score from claimed routes, completed destinations, and the longest continuous path!

Arrow Keys: Navigate menu options and select cards/routes | Space: Confirm selection or action | Z: Cancel current action or go back | Shift: Toggle between draw cards/claim route/draw destinations actions | Enter: Start game from menu | Esc: Pause/unpause game | R: Restart game (returns to start screen)
