Brick Break - Tetris Puzzle

Brick Break is a classic block-stacking puzzle game where tetrominoes (four-block shapes) fall from the top of the playfield. Your goal is to arrange these falling pieces to create complete horizontal lines, which will clear and award points. The game speeds up as your score increases, making it progressively more challenging. Stack blocks efficiently and prevent them from reaching the top to achieve the highest score possible!

Controls: Arrow Left/A: Move left | Arrow Right/D: Move right | Arrow Down/S: Soft drop | Arrow Up/W/Space: Rotate | Shift: Hard drop | ESC: Pause | R: Restart | ENTER: Start
