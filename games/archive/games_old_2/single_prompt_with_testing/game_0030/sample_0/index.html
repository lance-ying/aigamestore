FLOTTENMANÖVER

FLOTTENMANÖVER is a tactical naval combat game where you command a fleet and strategically attack enemy ships on a grid battlefield. Your objective is to sink all 5 enemy ships before they sink yours. Ships are hidden until hit, requiring careful deduction and strategy. Use special commander abilities to gain tactical advantages like sonar scanning or multi-shot salvos. Earn resource points each turn to unleash devastating abilities. Victory requires both tactical thinking and resource management.

Controls: Arrow Keys/WASD: Navigate | SPACE: Fire | Z: Cycle Abilities | SHIFT: Use Ability | ENTER: Start | ESC: Pause | R: Restart
