Detective's Eye: Hidden Objects

Find all hidden objects in each scene before time runs out! Search through increasingly cluttered environments - from a tidy study to a chaotic attic. Click on objects from the list at the bottom of the screen. Be careful: wrong clicks cost you precious seconds! Use hints wisely when you're stuck. Complete all 3 levels to win the game.

ENTER: Start game from menu | ARROW KEYS: Pan the camera view (Up/Down/Left/Right) | Z: Zoom in on the scene | SHIFT: Zoom out | SPACE: Use a hint (limited per level) | ESC: Pause/Unpause game | R: Return to start menu
