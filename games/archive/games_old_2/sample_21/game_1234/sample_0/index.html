Archero

Welcome to Archero! Battle through 5 challenging chapters filled with dangerous enemies. When standing still, you automatically attack the nearest foe. Move to dodge incoming attacks, but you'll stop shooting while moving. Clear all enemies in each room to progress. After each room clear, choose one of three powerful skills to enhance your abilities. Collect gold to purchase permanent upgrades between runs. Can you conquer all 5 chapters and defeat the final boss?

Arrow Keys / WASD: Move your character (Up, Left, Down, Right)
Space: Select first skill option during skill selection
Z: Select second skill option during skill selection
Shift: Select third skill option during skill selection
Enter: Start game from main menu
Escape: Pause/Unpause game
R: Return to main menu from game over screen
