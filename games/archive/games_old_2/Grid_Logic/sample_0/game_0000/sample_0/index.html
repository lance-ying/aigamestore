Grid Logic

Grid Logic is a classic Sudoku puzzle game. Fill the 9x9 grid so that each row, column, and 3x3 box contains the digits 1-9 exactly once. Start with pre-filled numbers and use logic to complete the puzzle. Choose from four difficulty levels: Easy, Medium, Hard, and Expert. Earn points for correct placements, but beware - five incorrect moves and it's game over!

Arrow Keys: Navigate between cells | 1-9: Enter number in selected cell | Space: Toggle pencil mark mode (add small candidate numbers) | Shift: Clear selected cell | Z: Undo last action | Enter: Start game / Confirm selections | Esc: Pause / Resume game | R: Restart (return to menu)
