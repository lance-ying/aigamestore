Old Man's Journey

Guide an old man through beautiful landscapes by shaping the terrain beneath his feet. The old man walks automatically along connected paths. Your task is to manipulate terrain layers by dragging them up or down to create continuous walkable surfaces.

Controls: Arrow Keys (←/→): Select terrain layer | Arrow Keys (↑/↓): Raise or lower layer | Space: Let old man walk | R: Restart level | Enter: Start/Continue | ESC: Pause
