Red's First Flight

Launch birds from a slingshot to destroy structures and eliminate all pigs! Drag the bird back to aim and set power, then release to fire. Complete levels by eliminating all pigs with limited birds. Earn up to 3 stars based on your score. Progress through increasingly challenging levels with different structures and defenses.

Controls: Arrow Keys (Left/Right): Adjust slingshot aim angle | Arrow Keys (Up/Down): Adjust slingshot power | Space: Launch the bird | R: Restart (available on game over screen) | Enter: Start game (from start screen) | Esc: Pause/Unpause game
