Danganronpa: Trial of Truth

Welcome to Danganronpa: Trial of Truth! You are a student trapped in a deadly game where you must solve murder mysteries to survive. During the Nonstop Debate phase, statements scroll across the screen with highlighted weak points (contradictions). Use your collected Truth Bullets (evidence) to shoot at these weak points to expose the lies. You have 5 Influence Points - losing all means Game Over. Successfully expose all contradictions to win the trial and escape!

Arrow Keys (←/→): Select Truth Bullet from inventory | Space: Fire selected Truth Bullet at weak point | Z: Absorb statement as new Truth Bullet (when available) | Enter: Start game from title screen | Esc: Pause/Unpause game | R: Return to title screen
