Fun with Ragdolls

Create and experiment with physics-based chaos! Build elaborate contraptions using ragdolls, cannons, mines, fans, and other interactive objects. Watch as ragdolls interact with your creations through various AI behaviors. There's no winning or losing—just pure creative experimentation with realistic physics. Place objects, adjust their properties, spawn ragdolls with different behaviors, and observe the hilarious results!

ENTER: Start the game | ESC: Pause/Unpause | R: Restart | Arrow Keys: Navigate build menu and adjust properties | SPACE: Place selected object or spawn ragdoll | 1-5: Quick select object types (1=Ragdoll, 2=Cannon, 3=Mine, 4=Fan, 5=Wall) | D: Delete mode toggle | Z: Clear all objects
