Harbor Merge Mystery

Welcome to Harbor Merge Mystery! You've inherited a dilapidated harbor restaurant and must restore it to its former glory while uncovering the island's secrets. Merge identical items on the grid to create higher-level dishes, then fulfill customer orders by dragging completed items to the Order Zone. Progress through 5 challenging levels, each bringing you closer to the truth. Watch out - if your board fills up with no possible merges, it's game over!

ENTER: Start the game from the main menu | ARROW KEYS: Navigate through grid cells (for keyboard control mode) | SPACE: Select/drop item at cursor position (keyboard mode) or toggle pause | ESC: Pause/unpause the game | R: Restart and return to main menu | Z: Confirm item merge or order fulfillment (keyboard mode)
