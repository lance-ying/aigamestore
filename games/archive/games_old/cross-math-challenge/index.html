CrossMath Challenge

CrossMath Challenge is a mathematical puzzle game where you must fill empty cells in a grid to create correct equations both horizontally and vertically. Each puzzle contains pre-filled numbers and operators (+, -, *, /). Your goal is to complete all four progressively challenging levels by solving each grid puzzle correctly. Remember: equations follow the standard order of operations (multiplication and division before addition and subtraction). You have 2 attempts per level - use them wisely!

Arrow Keys: Navigate between empty cells | Number Keys (1-9): Enter numbers into selected cell | Space: Submit puzzle for validation | Backspace: Clear current cell | Shift: Use hint (reveals one cell, -100 points) | Enter: Start game / Continue to next level | Escape: Pause / Unpause game | R: Restart (return to start screen)
