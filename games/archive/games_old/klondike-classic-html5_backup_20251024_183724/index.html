Klondike Classic HTML5

Klondike Classic HTML5 is a card game where you must move all 52 cards to four foundation piles, building them up by suit from Ace to King. Cards in the tableau can be moved between columns in descending rank and alternating colors. Draw cards from the stock pile when needed. Progress through 4 levels with increasing difficulty, from simple 1-card draws to timed challenges with 3-card draws. Use hints and unlimited undos to help solve each puzzle!

Arrow Keys: Navigate menus and UI (if applicable)
Space: Draw card(s) from stock pile
Z: Undo last move
Shift: Request hint (highlights a valid move)
Enter: Start game / Continue to next level
Escape: Pause / Unpause game
R: Restart (return to start screen)
