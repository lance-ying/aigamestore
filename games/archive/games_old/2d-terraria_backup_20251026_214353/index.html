2D Terraria

2D Terraria is a simplified sandbox adventure game where you mine resources, craft items, and battle enemies. Explore a procedurally generated world with day/night cycles, gather wood and stone, craft tools and build structures to survive. Defeat enemies, mine deeper for better resources, and ultimately defeat the final boss to win the game.

LEFT ARROW: Move left
RIGHT ARROW: Move right
UP ARROW: Jump
DOWN ARROW: Duck (when on ground) or drop through platforms
SPACE: Use current tool/weapon
Z: Switch between tools (pickaxe, axe, sword)
SHIFT: Place blocks (when holding blocks)
ENTER: Start game from title screen
ESC: Pause game
R: Restart game when on game over screen
