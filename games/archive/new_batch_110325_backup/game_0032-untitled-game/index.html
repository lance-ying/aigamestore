Kingdom Defense

Defend your kingdom against waves of invading enemies! Strategically place towers on fixed plots along the enemy path to stop them from reaching your base. You have 20 health points - don't let too many enemies through! Upgrade your towers and deploy powerful heroes with special abilities to turn the tide of battle. Complete waves to earn gold and unlock new defenses. Can you survive all the waves and protect your realm?

Arrow Keys (←→): Select tower type or navigate UI | Space: Place selected tower / Activate hero ability | Z: Upgrade selected tower | Shift: Deploy/Move hero to target location | Number Keys (1-4): Quick select tower types | Enter: Start game | ESC: Pause/Unpause | R: Restart (from game over screen)
