Subway Dash

Subway Dash is an endless runner where you control a character dashing through train tracks. Dodge oncoming trains, jump over barriers, slide under obstacles, and collect coins to increase your score. The game gets progressively faster and more challenging the longer you survive. How far can you run before getting caught?

LEFT ARROW: Move to the left lane
RIGHT ARROW: Move to the right lane
UP ARROW: Jump over obstacles
DOWN ARROW: Slide under barriers
ENTER: Start the game
ESC: Pause/unpause the game
R: Restart the game
