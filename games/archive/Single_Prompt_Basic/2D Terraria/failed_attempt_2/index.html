Pixel Miner

Pixel Miner is a 2D sandbox adventure where you explore a procedurally generated world. Mine resources, craft tools, build structures, and survive the night when monsters emerge. Gather enough resources to craft the ultimate item - the Portal - to win the game, but beware of the increasing difficulty as you progress!

Left Arrow: Move left
Right Arrow: Move right
Up Arrow: Jump
Down Arrow: Duck/Descend platforms
Z: Mine/Attack
Space: Place blocks/Use items
Shift: Switch between inventory items
Enter: Start game (at start screen)
Esc: Pause game
R: Restart game (at game over screen)
