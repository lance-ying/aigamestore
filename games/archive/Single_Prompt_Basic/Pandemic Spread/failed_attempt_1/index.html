Pathogen Pandemic

Pathogen Pandemic is a strategy game where you control a deadly pathogen trying to infect the entire world. Evolve your pathogen by spending DNA points to improve transmission, symptoms, and resistances while racing against scientists developing a cure. Infect all countries before the cure is completed to win, but be careful - if the cure finishes first or all infected countries are completely isolated, you lose!

Arrow Keys: Navigate between upgrades in the menu
SPACE: Select/purchase an upgrade when highlighted
Z: Toggle between world map and upgrade menu views
SHIFT: Accelerate game time (2x speed)
ENTER: Start game from title screen
ESC: Pause/unpause game
R: Restart game when on game over screen
