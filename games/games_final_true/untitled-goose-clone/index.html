Become a mischievous goose! Ruin the gardener's day by stealing items, setting pranks, and honking. Complete your tasks to win, but avoid the chasing gardener.

Arrow Keys / WASD: Waddle (Move) | Shift (hold): Run | Space: HONK! (Startles the gardener) | Z: Grab / Drop item (Interact) | Enter: Start Game | Escape: Pause / Resume | R: Restart (Game Over screens)
