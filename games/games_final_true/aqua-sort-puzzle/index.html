Sort colored liquids into tubes until each tube holds a single color.
Pour from a source tube to a compatible destination, matching colors or using empty tubes.
Complete all levels strategically to win!

Arrow Left/Right: Navigate between tubes | Space: Select tube to pour from, then select destination to pour into | Z: Undo last move | ESC: Pause/Resume game | Enter: Start game (from start screen) | R: Restart (returns to start screen from game over)
