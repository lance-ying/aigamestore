Pocket all colored balls using the cue ball to clear the table. Aim, adjust power, and avoid fouls like scratching. Master your pool skills across 9 challenging levels!

Arrow Left/Right: Rotate cue stick to aim (hold for continuous)
Arrow Up/Down: Adjust shot power (hold for continuous)
Shift + Arrow Keys: Apply spin (Up=top, Down=back, Left/Right=side)
Space: Execute shot
Z: Confirm cue ball placement (after foul)
Enter: Start game
Escape: Pause/Unpause
R: Restart to main menu
