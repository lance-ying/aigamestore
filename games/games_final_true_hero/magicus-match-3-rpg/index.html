A Match-3 RPG. Match elemental runes to cast spells, defeat monsters, and clear stages to win. Lose if your HP drops to zero.

Arrow Keys: Move the cursor on the grid | Z: Select a rune to swap / Confirm swap | Space: Cast Ultimate Spell (when Mana is full) | Shift: View Rune info (hold) | ESC: Pause the game | ENTER: Start the game | R: Restart the game (from Game Over screens)
