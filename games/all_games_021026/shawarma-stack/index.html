Run a shawarma restaurant, fulfilling customer orders quickly and accurately. Add exact ingredients, then serve. Earn points and maintain reputation across 5 levels to become a Shawarma Legend!

Arrow Keys: Navigate menus and select sauces (Up: Garlic Sauce, Down: Tahini Sauce) | Z: Add Meat | Space: Serve Order | ENTER: Start Game / Confirm | ESC: Pause / Unpause | R: Restart (from Game Over screen) | Shift: Quick pause toggle during gameplay
