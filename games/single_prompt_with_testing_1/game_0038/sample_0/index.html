SMALL WORLD

Command fantasy races to conquer territories on a hex-grid map! Select a race-ability combination, deploy tokens to adjacent territories, and accumulate victory points over 8 rounds. Conquer empty lands (costs 2 tokens) or occupied ones (costs 2 + enemy tokens). Put races into decline to score residual points while activating a new race. Compete against AI opponents to become the supreme conqueror!

Controls:
Arrow Keys: Navigate/select territories and race combinations
Space: Confirm selection (deploy tokens, choose race combo, end turn)
Shift: Put current race into decline
Z: Undo last token placement (before confirming)
ESC: Pause/unpause game | R: Restart to main menu | Enter: Start game
