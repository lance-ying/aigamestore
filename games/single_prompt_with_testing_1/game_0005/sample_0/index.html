Poly Bridge

Build bridges to guide vehicles across gaps in this physics-based puzzle game. Construct bridges using different materials (road, wood, steel, springs) by placing segments between anchor points. Each material has different properties and costs. After building, test your bridge by starting the simulation. Complete the level when all vehicles safely cross without the bridge collapsing, staying within budget and piece limits.

Arrow Keys: Move cursor in build mode | Space: Place/Connect node in build mode | A/D: Cycle through materials (Road, Wood, Steel, Spring) | S: Start simulation/Test bridge | R: Restart level (from game over) | Enter: Start game (from start screen) | ESC: Pause/Unpause
