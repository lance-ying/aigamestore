Sven's SudokuPad

Sven's SudokuPad is a classic Sudoku puzzle game where you must fill a 9×9 grid with digits 1-9. Each row, column, and 3×3 box must contain all digits exactly once. Start with an easy puzzle featuring helpful hints and clear visuals. Select cells using arrow keys, enter numbers with digit keys, and toggle between solution mode (fills the cell) and pencil mark mode (adds small candidate numbers). Complete the puzzle by correctly filling all empty cells!

Arrow Keys: Move cell selection up/down/left/right | 1-9 Keys: Enter digit in selected cell (solution mode) or toggle pencil mark (pencil mode) | Space: Toggle between solution mode and pencil mark mode | Shift: Clear the selected cell | Z: Undo last move | ENTER: Start game from start screen | ESC: Pause/unpause game | R: Return to start screen
