When a ball rolls slowly toward a pocket, it can stop just outside the capture threshold and never get pocketed. The current thresholds are not generous enough for very slow balls (speed < 0.5) and especially for nearly-stopped balls (speed < 0.1).

- Files affected: physics.js

  1. Add a special, very generous threshold for nearly-stopped balls (speed < 0.2)
  2. Increase the threshold for slow balls (speed < 1.0) significantly
  3. Make the capture zone larger overall to ensure slow balls get pocketed
  4. Add better handling for balls that are essentially stationary near pockets
