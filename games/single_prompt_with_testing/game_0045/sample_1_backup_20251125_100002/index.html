🎮 NEON WHITE - Speedrun FPS

OBJECTIVE: Eliminate all demons as fast as possible!

🎯 CONTROLS

WASD Keys: Move (Forward/Left/Backward/Right)

Arrow Keys: Rotate Camera (Look around)

SPACE: Jump (double-tap with card for super jump)

SHIFT: Dash (consumes card)

Z: Shoot (consumes card)

ENTER: Start Game

ESC: Pause

R: Restart

💡 STRATEGY

• Collect Soul Cards for weapons and abilities

• Cards can be used as weapons OR parkour abilities

• Find the fastest route using parkour shortcuts

• Chain abilities for maximum speed
