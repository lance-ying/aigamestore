<analysis>
- Issue identified: The user wants to build longer road segments (200px) but finds that doing so (specifically spanning the gap with just two segments) creates an unrealistically strong bridge that doesn't collapse under load. This is due to the `ROAD` material having a short `maxLen` (preventing 200px segments normally, though the user implies they want to do this) and a `strength` value that is too high relative to the span, preventing breakage.
- Files affected: `globals.js`
- Changes needed: 
    1.  Update `MATERIALS.ROAD.maxLen` from 100 to 200 to allow longer segments.
    2.  Decrease `MATERIALS.ROAD.strength` (e.g., from 50 to 30) to make it more susceptible to breakage, especially on long unsupported spans.
    3.  Increase `MATERIALS.ROAD.weight` (e.g., from 3.0 to 5.0) to increase the gravity load on the bridge, ensuring long spans sag and break under their own weight or the car's weight.
</analysis>
