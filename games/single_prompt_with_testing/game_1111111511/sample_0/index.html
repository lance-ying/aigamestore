NEON STACK

Neon Stack is a high-precision architectural arcade game. A block slides horizontally across the screen, and your objective is to stop it precisely on top of the block below. Any overhang is sliced off, causing the tower to become narrower and harder to build. Stack as high as you can to achieve a high score. Reaching 50 blocks constitutes a Master Win!

ENTER: Start | SPACE: Place Block | ESC: Pause | R: Restart
