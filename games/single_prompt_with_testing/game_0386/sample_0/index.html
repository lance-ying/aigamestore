LYNE

LYNE is a minimalist puzzle game where you connect matching shapes by drawing paths through the board. Each shape has a number indicating how many connections it needs. Fill all required connections to complete the puzzle. Features multiple levels with increasing complexity, elegant visuals, and a calming yet challenging experience.

Arrow Keys: Move cursor between nodes | Space: Select/deselect node to start/end a path | Z: Undo last connection | R: Restart puzzle (on game over screen) | ESC: Pause/unpause game | Enter: Start game
