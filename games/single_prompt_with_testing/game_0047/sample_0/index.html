Return of the Obra Dinn

You are an insurance investigator dispatched to the mysteriously returned merchant ship Obra Dinn. The crew has vanished, and the ship is adrift with only scattered clues remaining. Navigate the ship's deck using a first-person view, discover 8 hidden clues about the crew's fate, and piece together what happened. Beware of hostile spirits that haunt the vessel - they will chase you if they spot you. Find all clues and escape through the exit portal to complete your investigation.

Arrow Keys: Move forward/backward and turn left/right • Space: Interact with clues and exit portal • Shift: Sprint (move faster, but makes more noise) • ESC: Pause/Unpause game • R: Return to start screen • ENTER: Start game from title screen
