Arcadia Sports

Welcome to Arcadia Sports! Choose from 12 exciting sports mini-games, each offering unique arcade-style challenges. Test your skills in basketball shooting, racing, soccer penalty kicks, baseball batting, and more. Each game features dynamic difficulty that increases as you play, pushing you to achieve higher scores. Complete objectives, set personal bests, and master all sports to become the ultimate arcade champion!

Arrow Keys: Navigate menu selections, control player movement in mini-games | Space: Confirm selection, perform primary action (shoot, swing, jump) | Z: Secondary action (sprint, special move) | Shift: Modifier for advanced techniques | Enter: Start game from main menu | ESC: Pause/unpause during gameplay | R: Return to start screen from game over
