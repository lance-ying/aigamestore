Spelunky Cave Explorer

Explore dangerous procedurally-generated caves filled with treasures, traps, and enemies! As a daring spelunker, descend through multiple levels collecting gems and gold while avoiding deadly hazards. Use bombs to blast through obstacles and ropes to navigate carefully. Defeat or avoid monsters, discover secret areas, and try to reach the exit alive. Each run is unique with randomized layouts - how far can you delve into the depths?

Arrow Left/Right: Move left and right
Arrow Up: Climb ladders, enter doors, use rope
Arrow Down: Crouch, descend through platforms
Space: Jump
Z: Throw bomb (if available)
Shift: Use rope (if available)
