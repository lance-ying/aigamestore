Before Your Eyes

Experience a touching journey through life's most precious memories. You are a soul reliving key moments from your past, but you can only stay in each memory for a limited time. Press SPACE to "blink" and transition between memories, collecting emotional fragments to understand your story. Navigate through childhood, first love, artistic dreams, and heartbreaking truths. Reach the end of your journey before time runs out to find peace in the afterlife.

Arrow Keys: Navigate through memory scenes (left/right to move, up to interact with objects) | SPACE: Blink to advance through memories or interact with key moments | Z: Hold to reflect on current memory (slows time passage) | ENTER: Start the game | ESC: Pause/Unpause | R: Restart to main menu
