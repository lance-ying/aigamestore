FactoryBlocks

Welcome to FactoryBlocks! You're a factory engineer building automated production lines for alien overlords. Place conveyor belts and processors to transform raw materials into required products. Complete increasingly complex manufacturing challenges while managing limited space and resources. Master the art of efficient factory design to survive!

Arrow Keys: Navigate and move cursor in build mode | Space: Place/Remove selected component | Z: Cycle through available components (Conveyor, Processor, Rotator) | Shift: Toggle build mode on/off | Enter: Start game from menu | ESC: Pause/Unpause game | R: Restart and return to main menu
