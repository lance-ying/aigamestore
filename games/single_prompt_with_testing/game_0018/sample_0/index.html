TO THE MOON - Memory Traversal

Journey through Johnny's fading memories as Dr. Rosalene and Dr. Watts. Navigate fragmented memory rooms, collecting memory orbs to unlock deeper layers of Johnny's past. Each memory reveals a piece of his story and brings you closer to understanding his final wish—to go to the moon. Avoid temporal distortions that threaten to corrupt the memories.

Arrow Keys: Move • Space: Interact • Z: Temporal Stabilizer • Shift: Sprint • Enter: Start • Esc: Pause • R: Restart
