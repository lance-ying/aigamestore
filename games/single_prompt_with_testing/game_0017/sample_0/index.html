Pixel Wizard

Descend into a mystical cavern where every pixel is alive with elemental magic. As a powerful wizard, you must navigate through procedurally generated chambers filled with enemies and destructible terrain. Collect spell components to craft powerful magic combinations - burn through obstacles, freeze enemies, or create explosive chain reactions. Each element interacts realistically: water extinguishes fire, fire melts ice, and explosions scatter everything. Survive the descent, defeat enemies, and reach the deepest chamber to claim victory. Death is permanent, but knowledge persists.

Arrow Keys: Move left/right, fall through platforms | Space: Cast primary spell | Shift: Cast secondary spell (when unlocked) | Z: Switch active spell component | ENTER: Start game | ESC: Pause/Unpause | R: Return to start screen
