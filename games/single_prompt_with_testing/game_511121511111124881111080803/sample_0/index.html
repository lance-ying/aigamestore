BIT BLASTER XL

Bit Blaster XL is a retro arcade space shooter where you pilot a constantly moving ship through waves of enemies. Your ship auto-fires, so focus on dodging bullets, collecting power-ups, and destroying enemies to rack up high scores. Survive as long as possible in this intense bullet-hell experience inspired by classic arcade games!

Arrow Left: Turn ship counter-clockwise | Arrow Right: Turn ship clockwise | Arrow Up: Boost forward | Arrow Down: Brake | Space: Activate special weapon | ENTER: Start game | ESC: Pause/Unpause | R: Restart
