FIREWORK: Village Massacre Investigation

You are Lin Lixun, a rookie police officer investigating a mysterious massacre case in a remote mountain village. Navigate through dark corridors, collect evidence scattered throughout the village, and solve the case by finding all clues. But beware - supernatural forces lurk in the shadows, and time is running out. Collect all 8 pieces of evidence while avoiding the wandering spirits to uncover the truth behind the tragedy.

Arrow Keys: Move Lin Lixun in four directions | Space: Interact with evidence/clues when nearby | Shift: Sprint (move faster, but drains stamina) | Z: Use flashlight to reveal hidden paths (limited battery) | Enter: Start game from menu | ESC: Pause/Unpause game | R: Return to start screen
