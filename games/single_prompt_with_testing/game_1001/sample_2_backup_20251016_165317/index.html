DOT CONNECT

Connect adjacent dots of the same color to clear them from the grid! Draw lines horizontally or vertically between matching dots. Create a square with four dots of the same color to trigger a powerful explosion that clears ALL dots of that color. Complete level objectives within the move limit to progress. Each level has unique goals like clearing specific dot counts or reaching target scores.

Arrow Keys: Navigate menu selections | WASD Keys: Alternative navigation (not used in gameplay) | Space: Select/confirm menu option | ENTER: Start game from start screen | ESC: Pause/unpause game | R: Restart to start screen from game over
