The collision detection is flawed. When checking X and Z movements independently, the game prevents diagonal movement near walls, even when the player should be able to slide along the surface. This creates an unnatural movement restriction that doesn't match player expectations. I need to implement a more sophisticated collision detection method that allows sliding and maintains player momentum when approaching walls at an angle.
