A high-speed arcade racing game. Compete against AI on a procedurally generated city track. Drift through corners, use power-ups, and race 3 laps to win the Gold Cup.

Arrow Up / W: Accelerate | Arrow Down / S: Brake / Reverse | Arrow Left / A: Steer Left | Arrow Right / D: Steer Right | Space: Use Item / Horn | Shift: Drift (Hold while turning) | Enter: Start Game | Esc: Pause Game | R: Restart Game
