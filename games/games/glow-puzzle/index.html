Glow Puzzle

Connect all glowing dots on the grid by drawing a continuous path without lifting your finger or retracing any line segment. Each puzzle has exactly one solution - an Eulerian path that visits every connection once. Start at any dot and carefully plan your route. Complete increasingly complex puzzles to progress through all levels. Think ahead, as one wrong move can force you to restart the puzzle!

Arrow Keys: Navigate between dots | Space: Select/confirm dot and draw connection | Shift: Undo last connection | Z: Reset current puzzle | Enter: Start game | Esc: Pause/unpause | R: Return to start screen
