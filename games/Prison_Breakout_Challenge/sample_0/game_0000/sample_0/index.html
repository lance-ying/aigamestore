Prison Breakout Challenge

Escape from prison within 3 minutes! Navigate through 3 challenging levels, each with guards patrolling the area. Find keys, disable security systems, and unlock doors to reach the exit. Stay out of the guards' sight - if they spot you, they'll chase you down. Get caught and it's game over! Complete all objectives and escape before time runs out to advance to the next level. Can you break out of all three levels?

A/D or Left/Right Arrow: Move left/right | Space: Jump | Z: Interact with objects | Enter: Start game | Esc: Pause/Unpause | R: Restart to main menu
