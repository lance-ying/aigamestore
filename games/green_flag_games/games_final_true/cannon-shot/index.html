Cannon Shot

Position ramps and deflectors to guide cannon balls into buckets. Use limited balls to fill all buckets and win each level!

Controls:
Arrow Keys (←→): Select object (when not grabbed)
Space: Pick up/drop object (toggle grab mode)
Arrow Keys (↑↓←→): Move grabbed object position
A/D: Rotate grabbed object
Enter: Start game from menu
Esc: Pause/unpause game
R: Restart from game over screen
Z: Fire cannon (when in playing phase and not editing)
