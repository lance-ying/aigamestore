- Issue identified: The Matter.js physics engine continues to update every frame regardless of game state. When the game is paused, `Engine.update()` is still called, causing physics simulation (gravity, velocity, collisions) to continue running even though the game appears paused.

- Files affected: game.js

- Changes needed: Move the `Engine.update()` call to only execute when `gameState.gamePhase === "PLAYING"`. This will completely freeze all physics simulation when the game is paused, making time truly stop.
