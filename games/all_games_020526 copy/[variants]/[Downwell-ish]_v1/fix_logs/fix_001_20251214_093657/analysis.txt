This leads the player back to the initial "press enter to begin" screen, which is not the desired behavior for an auto-restart. The user expects to go directly into "PLAYING" mode.
- Files affected: `game.js`

This will bypass the start screen for automatic restarts, while preserving the manual 'R' key restart behavior (which explicitly sets the phase to "START").
