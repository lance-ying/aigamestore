A 2D action RPG. As a shadow ninja, fight through levels, upgrade skills, and defeat the Shadow Lord to save the realm.

Arrow Left/Right: Move character | Space / Arrow Up: Jump | Arrow Up/Down: Climb ladders | Arrow Down + Space: Drop through thin platforms | Z: Attack (Melee) | Shift: Dodge / Dash | Down + Z: Special Skill 1 (Ground Smash) | Up + Z: Special Skill 2 (Rising Strike) | ESC: Pause Game | R: Restart Game (from Game Over/Win screens) | ENTER: Start Game
