- Issue identified: The `RangeError: Invalid count value: -1` occurs because the `String.repeat()` method is being called with a negative argument. This happens when `gameState.lives` or `gameState.bombs` fall below zero, which can occur due to game logic (e.g., player taking damage when lives are 0, or using a bomb when bombs are 0). The `renderHUD` function then attempts to render a negative number of symbols.
- Files affected: `ui.js`
- Changes needed: Modify the `renderHUD` function to ensure that `Math.max(0, ...)` is used when determining the count for `String.repeat()` for both lives and bombs, preventing negative values from being passed to the method.
