The user wants to make sure that when the game is paused, the HUD (Heads-Up Display) is not hidden. Let me analyze the code to see what happens when the game is paused.

Looking at `ui.js`, the `renderUI()` function handles what gets displayed. Let me check what happens in each game phase:

1. `START` phase: Shows start screen with instructions
2. `PLAYING` phase: Shows HUD with score and health bar
3. `PAUSED` phase: Not explicitly handled! This is the issue.
4. `GAME_OVER_LOSE` phase: Shows game over screen

So when the game is paused, there's no case for `gamePhase === 'PAUSED'`, which means the canvas is just cleared and nothing is drawn. This would hide the HUD.

Looking at `input.js`, I can see the pause logic:
- When ESC is pressed, it toggles `input.pause`
- In `processInput()`, when `input.pause` is true and phase is `PLAYING`, it changes to `PAUSED`
- When `input.pause` is false and phase is `PAUSED`, it changes back to `PLAYING`

The fix is simple: In `ui.js`, I need to add a case for when `gamePhase === 'PAUSED'` that:
1. Shows the HUD (score and health bar) - same as PLAYING
2. Also shows a "PAUSED" message overlay

Let me implement this fix in `ui.js`:

```javascript
else if (gameState.gamePhase === 'PAUSED') {
    // Show HUD (same as PLAYING)
    ctx.textAlign = 'left';
    ctx.fillText(`Score: ${Math.floor(gameState.score)}`, 20, 30);
    
    // Health Bar
    const maxBarWidth = 200;
    const currentBarWidth = (gameState.player.health / gameState.player.maxHealth) * maxBarWidth;
    ctx.fillStyle = '#333';
    ctx.fillRect(20, 40, maxBarWidth, 20);
    ctx.fillStyle = '#ff3333';
    ctx.fillRect(20, 40, currentBarWidth, 20);
    ctx.strokeStyle = '#fff';
    ctx.strokeRect(20, 40, maxBarWidth, 20);
    ctx.fillStyle = 'white';
    ctx.font = '12px Arial';
    ctx.fillText(`HP: ${gameState.player.health}`, 25, 55);
    
    // Pause overlay with semi-transparent background and centered text
    ctx.fillStyle = 'rgba(0, 0, 0, 0.5)';
    ctx.fillRect(0, 0, canvas.width, canvas.height);
    ctx.fillStyle = 'white';
    ctx.font = 'bold 48px Arial';
    ctx.textAlign = 'center';
    ctx.fillText('PAUSED', canvas.width / 2, canvas.height / 2);
    ctx.font = '20px Arial';
    ctx.fillText('Press ESC to resume', canvas.width / 2, canvas.height / 2 + 50);
}
```

This approach keeps the HUD visible while adding a pause overlay that darkens the screen and displays pause instructions.
