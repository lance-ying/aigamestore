FruitDrop Fusion

FruitDrop Fusion is a physics-based puzzle game where you drop fruits into a container to create larger fruits through fusion. When two identical fruits collide and settle, they merge into the next tier fruit. Progress through 4 challenging levels with different goals and difficulty settings. Create strategic combos for bonus points and reach the ultimate goal: creating a Watermelon! Watch out for the lose line at the top - if any fruit crosses it, it's game over!

Controls: Arrow Left/A: Move drop position left | Arrow Right/D: Move drop position right | Space/Arrow Down/S: Drop the fruit | Enter: Start game/Restart from game over | Escape/P: Pause/Unpause game | R: Return to start screen
