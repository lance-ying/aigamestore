Papa's Freezeria To Go!

Papa's Freezeria To Go! is a time management game where you run an ice cream shop. Take customer orders and craft perfect sundaes by moving through different stations. Pour ice cream, add mix-ins, blend to the right consistency, and finish with toppings. Satisfy customers by matching their orders precisely to earn tips and build your reputation. Balance speed and accuracy as customers become impatient while waiting!

Left Arrow: Move left between stations
Right Arrow: Move right between stations
Up Arrow: Move up in the current station menu
Down Arrow: Move down in the current station menu
Space: Select/Confirm action at current station
Z: Cancel current action
Shift: Rush/Speed up certain actions
Enter: Start game (at start screen)
Esc: Pause/Unpause game
R: Restart game (at game over screen)
