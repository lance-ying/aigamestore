Elemental Alchemy

Discover the universe of creation! Start with four basic elements (Air, Earth, Fire, Water) and combine them to create new materials, objects, and concepts. Mix elements by selecting one with Arrow Keys and combining it with another using Space. Unlock categories like Life, Technology, and Magic as you progress. Your goal is to discover all 100+ unique elements and complete the creation tree!

Arrow Keys: Navigate through available elements | Space: Select/Combine elements (select first element, then select second to combine) | Enter: Start game from title screen | Esc: Pause/Unpause game | R: Return to start screen
