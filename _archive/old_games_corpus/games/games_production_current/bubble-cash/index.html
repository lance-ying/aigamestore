Bubble Cash

Welcome to Bubble Cash! Clear all the colored bubbles from the board before time runs out. Aim your bubble shooter to match 3 or more bubbles of the same color to pop them. Make strategic shots to create chain reactions and cause bubbles to detach and fall for bonus points. Complete all 3 increasingly challenging levels to win the game!

Arrow Left: Rotate shooter left
Arrow Right: Rotate shooter right
Space: Fire bubble
Z: Swap current bubble with next bubble
Enter: Start game (from start screen)
Esc: Pause/Unpause game
R: Restart (return to start screen)
