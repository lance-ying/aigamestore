Bring You Home

Guide Polo safely home by rearranging the world around him! Each level consists of several panels. Your job is to swap panels to create a safe route. The key strategy: move the EXIT panel forward to create a shorter safe path! Polo walks automatically, so plan carefully. Avoid spikes, enemies, and gaps. Only the START panel is fixed—you can move everything else, including the EXIT!

Controls: Arrow Keys (←/→): Move cursor between panels | Space: Select panel (select 2 panels to swap them) | Z: Start Polo walking | Shift: Rewind if Polo is in danger | Enter: Start game | Esc: Pause | R: Restart
