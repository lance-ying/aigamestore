Block Blast

Block Blast is a strategic puzzle game where you place different geometric blocks on a 9x9 grid. Your goal is to fill entire rows or columns to clear them and earn points. You have three blocks available at a time - use arrow keys to select a block, then position and place it on the grid. The game ends when you can't place any of the available blocks. Plan your moves carefully to maximize your score with combo bonuses for clearing multiple lines at once!

LEFT ARROW: Move selected block or cursor left
RIGHT ARROW: Move selected block or cursor right
UP ARROW: Move selected block or cursor up
DOWN ARROW: Move selected block or cursor down
SPACE: Place the selected block on the grid
Z: Cycle through available blocks
ENTER: Start the game from the start screen
ESC: Pause/unpause the game
R: Restart the game (works on game over screen too)
