DOT CONNECT

Connect adjacent dots of the same color to clear them from the grid! Use arrow keys to move the cursor, then press Space to select dots and build a path. Adjacent dots include horizontal, vertical, and diagonal neighbors. Create a square with four dots of the same color to trigger a powerful explosion that clears ALL dots of that color. Complete level objectives within the move limit to progress.

CONTROLS: Arrow Keys: Move cursor | Space: Select/extend dot at cursor | Enter: Confirm and clear selected path | ESC: Cancel selection or pause game | R: Restart from game over screen
