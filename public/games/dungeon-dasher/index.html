Dungeon Dasher

Navigate through 5 challenging levels of procedurally generated dungeon rooms. Move grid-by-grid to dodge enemies and projectiles. Stand still to auto-attack the nearest foe. Clear each room to earn powerful upgrades that stack throughout your run. Defeat increasingly difficult enemies including melee grunts, ranged shooters, heavy tanks, and a final boss. Survive to the end to claim victory!

Arrow Keys (↑↓←→): Move one grid cell in the corresponding direction | Spacebar: Confirm upgrade selection / Start game from menu / Resume from pause | Z: Pause/Unpause the game | Enter: Start the game from the title screen | Esc: Pause the game | R: Restart from game over screen (returns to title)
