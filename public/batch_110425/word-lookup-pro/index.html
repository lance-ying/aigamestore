Word Lookup Pro

Search through a massive dictionary to find valid words and create anagrams. Type letters to search for matching words, use wildcards (? for single letters, * for multiple) to find patterns, or switch to Anagram mode to discover all possible words from your letters. Each word has a point value based on letter scoring. Challenge yourself to find high-scoring words, discover rare combinations, and expand your vocabulary through various word lists and puzzle modes!

Arrow Keys: Navigate menus and word lists | Space: Toggle between Search and Anagram modes | A-Z Keys: Type letters into the search field | Shift: Toggle uppercase (visual only) | Z: Backspace/delete last letter | Enter: Start game from menu | Esc: Pause/unpause game | R: Return to start screen
