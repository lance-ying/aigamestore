Number Surge

Number Surge is an action-puzzle runner where you control a numbered block racing forward. Collect blocks with numbers equal to or smaller than yours to grow, but AVOID blocks with larger numbers - they'll end your run! Navigate the track, dodge hazards, and grow your number as large as possible. At the end of each level, you'll face walls with numbers - you can only break through walls if your number is greater than or equal to the wall's number. Break walls to earn bonus points!

Arrow Left/Right: Move horizontally along the track | Space: Select menu options / Continue after level complete | Shift: Pause/Unpause during gameplay | Enter: Start game from main screen | Esc: Pause game | R: Restart (return to start screen)
